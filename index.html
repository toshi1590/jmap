Jmap

Jmap jQuery plugin

日本地図 jQuery プラグイン - Released under the MIT license

Jmap jQuery pluginは、情報配信に適したシンプルな形状の日本地図を表示することができます。 出力される地図はHTMLコーダーやCSSデザイナーも気軽に利用できるようHTMLとCSSのシンプルな構造となっており、カスタマイズ性に優れかつ便利な機能を複数備えています。

1 クイックスタート - 最初の一歩

最初の一歩は驚くほど簡単です。

Start!

最初の一歩はJmapのサイト上からJmap jQuery pluginをダウンロードする所からはじめましょう。

Jmap 通常版 Jmap 圧縮版

ファイルのダウンロードが完了したらHTMLファイルを作成し、その中でjQueryと、ダウンロードしてきたJmap jQuery pluginを読み込みましょう。

jQueryとJmapの読み込みコード

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
<script type="text/javascript" src="./jmap.js"></script>

チュートリアル作成時のjQuery最新版は3.4.1です。最新のものを利用したい場合は以下リンクからURLを取得してください。

Google Hosted Libraries
Example

<html>
<head>
    <title>Jmap jQuery Example</title>
    <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>
    <script type="text/javascript" src="./jmap.js"></script>
    <script type = "text/javascript">
        $(document).ready(function() {
            // これ以降「Example」に記載するエリア
        });
    </script>
</head>
<body>
    <!-- id="xxx" の「xxx」部分は必要に応じて変更 -->
    <div id="jmap"></div>
</body>
</html>

Tips!

HTMLの要素に jmap メソッドを適用するだけで、デフォルトの地図が表示されます。

Example

$('#jmap').jmap();

2 ステップアップ　- カスタマイズ

その一味は大きな変化を生み出します。

Tips!

Jmapには地図をカスタマイズするのに必要な様々なパラメータが用意されています。

パラメーター
Example

$('#jmap').jmap({
    width: '100%',      // 横幅
    height: '450px',    // 縦幅
    lineColor: 'green', // 地図背景の枠線幅
    lineWidth: 2,       // 地図背景の枠線色
    backgroundColor: '#EFEFEF', // 地図背景の色
    backgroundRadius: '0.5rem',   // 地図背景の角丸
    backgroundPadding: '0.5rem',  // 地図背景の余白
    showPrefectureName: true,   // 都道府県名の有無
    prefectureNameType: 'full', // 都道府県名の表示形式
    prefectureBackgroundColor: '#BFEAB5',   // 地図の色
    prefectureBackgroundHoverColor: '#DFFFD5',  // 地図ホバー色
    fontSize: '0.6rem',     // フォントサイズ
    fontColor: '#467E36'    // フォント色
});

5 オリジナリティー - 独自設定

感性の赴くままにデザインしよう。

JmapはHTMLとCSSで構成された地図を出力します。もちろん自分が作成したCSSやJavascriptで直接操作する事ができます。

Class Name
クラス名		 Description
説明		 CSS Selector
セレクター
jmap-container Jmapのラッパーに付与されるクラス名です。 .jmap-container {}
jmap-pref Jmapの全都道府県に共通で付与されるクラス名です。 .jmap-container { background-color:#0f0; }
jmap-pref[jmap-pref="都道府県コード"] Jmapの各都道府県に都道府県コード別に付与されるクラス名です。 .jmap-pref[jmap-pref="都道府県コード"] { background-color:#f00; }
jmap-infobox Jmapの情報表示エリアに付与されるクラス名です。 .jmap-infobox { background-color:#f00; }
jmap-divider Jmapの遠島区分け線に付与されるクラス名です。 .jmap-divider { border-bottom-color:#f00; }
